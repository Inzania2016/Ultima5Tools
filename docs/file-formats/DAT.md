# DAT Format (Current Toolkit Understanding)

DAT appears to be a mixed family of files with varied internal semantics.

## Current implementation scope

- Metadata inspection only:
  - full path
  - basename
  - extension
  - file size
  - a conservative heuristic note

## Non-goals (for now)

- Pretending there is a single complete DAT schema.
- High-confidence semantic decode without evidence.
